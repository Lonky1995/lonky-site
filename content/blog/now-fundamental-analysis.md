## 核心结论

1. **基本面真好（数字层）**：营收 +20.9% YoY、FCF $4.6B（FCF/净利润 **262%**）。3 年 FCF $2.7B → $4.6B 加速增长。营业利润率 13.7% 偏低是因为 SBC + R&D 高投入，真实现金创造能力远超 GAAP。

2. **叙事重大转向（事件层）**：4/1 + 4/22 + 5/15 三份连续 8-K 揭示完整融资链——**$3B 循环信贷 + $4B Term Loan 过桥贷款 + $4B 五档高级债 = $11B 流动性储备**，全部用于 **$4B 收购 Armis Security**（物联网/OT 网络安全独角兽）。这是 NOW 史上最大并购，从 ITSM 工具升级为 **IT + OT 安全统一平台**。

3. **管理层 DNA 解释了收购逻辑（v3 新视角）**：CEO **Bill McDermott** 前 SAP CEO（2010-2019 市值翻 3 倍），典型增长型 + 战略并购打法；CFO **Gina Mastantuono** 推动信用评级从 BBB+ 升至 A-，"先备弹药再行动" 是她的手笔；**Amit Zavery** 三职合一（President+CPO+COO），负责 Armis 整合，是关键人物风险；创始人 **Fred Luddy** 仍在董事会，没反对这次大动作是合理性强信号。

4. **估值 Football Field（v3 新视角）**：7 个估值锚点共识区间 **$85 - $236**（中位 $137.50），当前 $95 **处于下端**——即使空头观点成立（Armis 整合失败 → 低端 $85），下行空间仅 **-11%**；上行空间 **+45%** 到中位、**+148%** 到高端。多空分歧本质是 **估值视角切换**（GAAP PE 56x vs P/FCF 21x），谁占主导决定方向。
